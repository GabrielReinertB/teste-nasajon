# teste-nasajon
Solução para a prova técnica NASAJON onde lê municípios de um CSV, enriquece com a API pública do IBGE (nome oficial, UF, região, código), gera resultado.csv e envia estatísticas calculadas para uma Edge Function Supabase via JWT. Matching fuzzy com rapidfuzz para tratar erros de digitação e acentuação.
